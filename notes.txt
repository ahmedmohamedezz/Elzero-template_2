  - visit: https://fonts.google.com/
  - choose font family & sizes
  - get html link

- how to use icons from font-awesome:
  - visit: https://fontawesome.com/download
  - download the free for web version
  - unzip the file
  - copy the `webfonts` directory, and paste it in your project
    ? put that directory in the same level as the css directory
  - copy the `all.min.css` file in your project & link to your html
    ? put it inside your css directory (it calls 'webfonts' with '../webfonts')
    ? so make sure you structured your project properly
